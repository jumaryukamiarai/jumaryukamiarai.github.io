📊 私たちの目標

組織の基本目標

  • みんなが楽しく継続的にお囃子ができること
  • 子供の世代まで持続可能な運営ができること

⚠️ 令和4年度時点の問題点

残高の大幅な減少

令和元年度期初: 102.6万円 → 令和4年度末: 73.5万円（約29万円の減少）

主な問題点

  • 令和元年以前から慢性的な財政赤字
    → 令和元年の一人当たり飲食費は高い（18,030円）、支出が収入を上回る状態
  • コロナ禍の収入減少
    → 令和2～4年度は行事が中止・縮小され、収入がストップ
  • 唯一の収入である花代の減少
    → 今まで、地域の皆様からの花代の収入のみで運営を続けてきたが、花代の収入は年々減少傾向
  • お囃子道具、倉庫の老朽化
    → 今後、多大な修繕費用が必要になると見込まれる
  • メンバーの未定着
    → 約10年前から身内以外のメンバーが新規に入らず、組織の活性化が課題

✨ 財政の健全化

💡 経費見直しの取り組み

令和4年度から新体制になり、令和5年度から以下の経費見直しを段階的に実施し、財政の健全化を進めてきました。

  • 飲食代の購入先の変更、外食の中止 - より経済的な選択へ
  • 備品や消耗品の購入先を見直し - より安いところを「調査⇒購入先切替」でコスト削減を実現
    • 例）花代返しタオル240本: 指田商会 48,620円 → たおるのみせ 27,120円（約44%削減、21,500円の節約）

💡 令和7年度の特殊事情

必要な投資項目

組織の継続と発展のため、以下の必要な投資を実施しました：

  • 半纏（13着）: 22.7万円 ← 新メンバー増加に対応
  • 半纏帯（13本）: 3.1万円
  • 弓張提灯: 5.5万円 ← 活動の充実化
  • 獅子舞修理（予定）: 5.0万円 ← 重要な道具の維持

投資合計: 約36万円

✨ ポイント

この必要な投資（36万円）がなければ、令和7年度も約2.1万円の黒字の予定です。

※通常の運営費は適切にコントロールできています

🎯 今後の方針

1. 見直しと改善を継続

新しいメンバーも増えているため、試行錯誤しながら最適な運営を模索します

2. 実施できる支出抑制を継続

  • 飲食費の効率的な運用
  • 必要な備品の計画的な購入

3. 道具を大事に扱うことの徹底

  • お囃子道具や倉庫の適切な維持管理 - 道具を大切に扱い、長く使えるよう心がける
  • 計画的な修繕 - 必要な修繕を適切なタイミングで実施
  • 高額な道具の組織購入 - 個人負担を軽減し、メンバー全員が活動しやすい環境を整備

※安定した財政基盤があれば、必要な投資を適切なタイミングで実施できます

4. メンバーの皆さんへのお願い

花代として一人当たり年間2,000円程度のご協力をお願いします

📌 まとめ

🔍 問題点

  • 令和元年度は赤字体質（一人当たり飲食費18,030円）
  • このままでは財政が悪化し、持続可能な運営が困難
  • 物価上昇により、対策なしでは令和7年度に資金が枯渇

✅ 改善策

  • 飲食代の購入先変更、外食の中止
  • 備品・消耗品の購入先変更
  • 一人当たり飲食費を約79%削減（令和6年度、18,030円→3,806円）
  • 3年間で約63.7万円の支出を抑制

📊 現在の状況

  • 通常運営は健全化を達成
  • 令和7年度は必要な投資（36万円）により一時的な赤字
  • 投資を除けば若干の黒字基調
  • 唯一の収入である花代は減少傾向が続く見通し
  • 新メンバー増加により活動が活性化（14人→26人）

🔮 今後の展望

  • メンバーに年2,000円の花代協力をお願い
  • 必要な投資を適切なタイミングで実施できる財政基盤を確保、緊急の修繕費にも対応
  • 子供たちの世代まで続く、持続可能な運営を実現
